# Script Guide

Scripts are grouped by purpose so the folder is easier to scan:

- `scripts/model_tools/`: model download, quantization, and LoRA training
- `scripts/eval/`: evaluation utilities
- `scripts/data/`: curriculum and demo-data generation
- `scripts/audio/`: child-speech augmentation helpers

Compatibility wrappers are kept at the top level of `scripts/` so existing commands still work.
